(자바스크립트) 2~n까지의 수에서 소수 구하기

조건)
소수(prime number)는 자신보다 작은 두 개의 자연수를 곱하여 만들 수 없는 1보다 큰 자연수입니다.
다시 말해서 1과 그 수 자신 이외의 자연수로는 나눌 수 없는 자연수를 소수라고 부르죠.
5는 2나 3, 4로 나눌 수 없기 때문에 소수입니다. 5를 이들 숫자로 나누면 나머지가 있기 때문이죠.
2부터 n까지의 숫자 중 소수만 출력해주는 코드를 작성해봅시다.
n = 10이라면 결과는 2,3,5,7이 되어야겠죠.
주의: 작성한 코드는 임의의 숫자 n에 대해 동작해야 합니다.


풀이)
let n = prompt("숫자를 입력해주세요");
outer:
for(let i=2; i<=n; i++){
  for(let j=2; j<i; j++){
    if(i%j==0) continue outer;
  }
  alert(i);
}

만약 n=10일 경우: 2,3,5,7 이 출력된다.

여기서 중요한 점은 중첩 for문에서 일반적인 continue가 아닌 label이 달린 continue를 쓴 이유다.
label 없이 continue를 쓰게 될 경우 외부 for문으로 돌아가는 것이 아닌 continue가 속해있는 반복문으로 돌아가게 된다. 
이럴경우 소수 출력을 위한 i값에서 해당 조건에 따른 값을 무시해야하는데 중첩된 반복문에서만 무시되므로 i 값에선 값이 그대로 사용된다.
예를 들면 i=4일 경우 소수가 아님에도 continue가 돌아가는 곳이 내부 반복문이므로 외부 반복문에서는 4를 무시하지 못하게 되고 출력하게 된다.
이 경우 n=10이면 2,3,4,5,6,7,8,9,10이 출력된다. 그렇기 떄문에 continue를 통해서 조건에 걸리는 값을 날리기 위해서 label이 붙은 continue를 통해
외부 반복문에서 조건에 걸리는 값을 날려줘야한다.
